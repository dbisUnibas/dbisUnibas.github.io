of the Databases and Information Systems research group, University of Basel

This is the github website of the Databases and Information Systems Research (DBIS) Group at the University of Basel. The main focus of the DBIS group, University of Basel, is the development of novel database concepts and their application to large-scale distributed information systems, in particular to Data Management in the Cloud, Multimedia Information Retrieval, and Service-oriented Architectures & Process-based Systems (Workflows). From a technical perspective, this includes aspects of distribution, reliability, scalability, mobility, heterogeneity, dynamic adaptability, and semantics. The results of this work are applied to various fields like Crowdsourcing applications, e-Health or e-Science.

With the tremendous increase of recording devices and the resulting abundance of digital images, audio files and videos, finding a particular one in ever-growing collections is more and more becoming a major challenge. Existing approaches to retrieve multimedia mostly still rely on text-based retrieval techniques to find desired sequences. With vitrivr, we present you a fully working system for indexing and retrieving image, audio, video and 3D-model data based on the content.

vitrivr consists of ADAMpro, Cineast and Vitrivr NG. Find more details on the corresponding pages. For more information, visit the project website: vitrivr.org

ADAMpro

ADAMpro is a database system to store and retrieve multimedia data. It provides Boolean retrieval and similarity search and makes use of a various indexing methods for efficient retrieval.

ADAMpro on GitHub
Cineast

Cineast is a multi-feature sketch-based content multimedia retrieval engine. It is capable of retrieving images, audio, videos and 3D-models based on sketches or example objects.

Cineast on GitHub
Vitrivr NG

Vitrivr NG is a web-based user interface for searching and browsing mixed multimedia collections.

Vitrivr NG on GitHub

ReqMan

ReqMan, short for Requirements Manager is a JavaFX application to manage (project) requirements. It can be used to track the progress of projects in various contexts. We currently use it primarely for the 'Programming Project' Lecture. The entire project is available on GitHub.

Virtual Exhibitions

The Virtual Exhibition system is a generic, customisable virtual reality system in which curators can easily create multi-modal, virtual 3D exhibitions which visitors can navigate through to enjoy the displayed exhibits. The back-end, as well as the VR front-end are available on GitHub.
